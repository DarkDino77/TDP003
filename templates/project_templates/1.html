Portfolio Website Overview

Introduction

This portfolio website serves as a comprehensive showcase of various projects developed by the team. It's built using Flask and Python and was created as part of a web course.

Structure

The website consists of four main sections: the Index Page, Project Search Page, Techniques Page, and Individual Project Pages.

Core Features

  1. Index Page: Serves as the landing page and introduces the developers.
  2. Project Search Page: Allows users to browse through the various projects that the developers have created.
  3. Techniques Page: Lists the different techniques used across all projects, and serves as a search filter for the Project Search Page.

Additional Elements

  • Project Stats: Each individual project page includes various statistics about the project.
  • Dynamic Search: The Project Search Page allows dynamic searching based on user-selected criteria.

Technologies Used

The website is primarily built using Python and Flask. It also utilizes HTML, CSS, and potentially JavaScript for frontend development.

Development Environment

The project was developed using a local development server and was tested for both desktop and mobile responsiveness.

It was part of a curriculum, thus adhering to specific course guidelines and milestones.
