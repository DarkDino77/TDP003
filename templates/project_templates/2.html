Python Programming Course Overview

Introduction

This Python programming course is designed for beginners and focuses on teaching the basics of Python through six practical lab exercises. It avoids the use of higher-order functions like classes, but allows the use of lambdas.

Course Structure

The course is structured into six practical "lab moments," each consisting of a few assignments that reinforce Python programming concepts.

Core Labs

  1. Lab 1: Introduction to Python syntax and basic data types.
  2. Lab 2: Control flow using if-else statements and loops.
  3. Lab 3: Functions and lambdas.
  4. Lab 4: File I/O and text manipulation.
  5. Lab 5: Lists and dictionaries.
  6. Lab 6: Final project and review.

Additional Elements

  • Grading Criteria: The course has specific grading criteria based on lab performance and final project.
  • Guidance: Each lab comes with an instructor-led explanation and example codes.

Technologies Used

The course strictly focuses on Python, with minimal use of additional libraries. It allows the use of lambdas but avoids other higher-order functions.

Who Is This Course For?

This course is intended for individuals who are new to programming or Python and wish to learn the basics without diving into advanced topics like classes.
